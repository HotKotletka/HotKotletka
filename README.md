# 💫 About Me:
Работаю QA Engineer. В зоне моей ответственности — тестирование интерфейсов, API и интеграций с внешними сервисами (CRM, B2B-системы и другие). Перед каждым релизом провожу полное регрессионное тестирование ключевых блоков, чтобы убедиться, что всё работает стабильно.<br><br>Участвую в исследовательском тестировании вместе с аналитиком — ищем нестандартные сценарии и проблемы, которые не видны на первый взгляд. Также анализирую метрики и обратную связь пользователей, чтобы понимать, что можно улучшить в интерфейсах, и проверяю эти гипотезы.<br><br>Занимаюсь поддержкой внутренних пользователей: помогаю разобраться с инцидентами, консультирую, настраиваю систему под их задачи, провожу обучение.<br><br>Пишу и поддерживаю автотесты на Python. Ответственно подхожу к задачам, готова учиться новому и повышать свой уровень в автоматизации и тестировании в целом.<br>
